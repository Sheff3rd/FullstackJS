FullstackJS
=====================

### About

Simple and ready to use out of the box pure JS server.
Just Fork the reposytory for your own needs

### Usage

```
npm install
npm start
```

Open 'http://localhost:3000/' in your browser!


### Eslint

To run eslint ``` npm run lint ```
